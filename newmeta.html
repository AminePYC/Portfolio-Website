About

From writing code
to breaking it apart

I started in software engineering — building systems, writing clean code, thinking about architecture. At some point I became more interested in how things break, how they get weaponised, and what happens inside a malicious binary when it runs. That pivot led me here.

I analyse real malware from MalwareBazaar, map threat actor infrastructure through OSINT pivoting, and document each step publicly — because working in the open is how you build credibility before you have a job title to back it up.
